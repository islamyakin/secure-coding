# Instalation

## With Container
- Create Folder secure-code
  ```bash
  mkdir secure-code
  cd secure-code
  ```
- Download Script
  ```bash
  wget https://raw.githubusercontent.com/islamyakin/secure-coding/main/sonarqube/docker-compose.yaml
  ```
- Run with docker compose
  ```bash
  docker compose up -d
   ```
